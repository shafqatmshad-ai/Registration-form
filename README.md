# Registration-form # 🎓 Student Registration Form

This is a simple **Student Registration Form Web Application** built using HTML, CSS, and JavaScript. It allows users to enter student details and submit them for storage or further processing (e.g., Google Sheets or backend database).

---

## 🚀 Features

- Student registration form UI
- Input fields for:
  - Name
  - Father Name
  - CNIC
  - Date of Birth
  - Gender
  - Class
  - Address
  - Result
  - Photo URL / Upload
- Clean and responsive design
- Easy to integrate with Google Sheets / backend API

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (optional for form handling)
